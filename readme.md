# Geex Task

**3 Sections**
- Routines — tasks that repeat daily, weekly, or on a custom interval
- Scratch pad — free-form thoughts, no structure
- Long-term goals — tasks with a deadline and repeat schedule until that date

**Pomodoro**
- Standard 25/5 preset + fully custom timer
- Tied to a specific task ideally

**Music Player**
- Paste YouTube/Spotify links and build a playlist
- Basic controls: play, skip, volume

**Reminders**
- Browser push notification + sound + on-screen alert
- Set per task or routine
